# measurements-I_V


Projeto measurements I_V/
│
├── main.py
├── botoes.py
├── graficos.py
├── tela.py
└── util.py

main.py: Carrega e coordena tudo, criando a janela, inicializando o gráfico e adicionando botões.
tela.py: Contém a função que cria a janela principal.
botoes.py: Gera e configura os botões e suas interações.
graficos.py: Contém as funções para plotar, limpar e exportar gráficos.
util.py: Pode ser usado para funções utilitárias comuns.
